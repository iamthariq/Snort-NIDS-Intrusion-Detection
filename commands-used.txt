RULE 1: TCP SYN Scan Detection

--------------------------------------------------

RULE 10: Directory Traversal Detection

RULE 11: Sensitive File Access Detection (.git)

RULE 12: SQL Scanning Detection (SQLMap)

RULE 13: Command Injection Attempt Detection
